**name type**

The type of a named entity.

This feature is present on objects of type [*lex*](otype).

It is a comma separated list of the following values:

---|---
`pers`   |person
`mens`   |measurement unit
`gens`   |people
`topo`   |place
`ppde`   |demonstrative personal pronoun
