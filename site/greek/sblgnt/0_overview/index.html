Introduction¶

This is the key to the meaning of the features of the sblgnt dataset.

By all means consult the original documentation by Randall Tan.

We organize the features in several groups, roughly analagous to the types of objects we have:

  • grid
  • sectional
  • word
  • linguistic
  • generic

Grid features¶

feature description examples
otype node type book verse clause phrase word
oslots slot containment 1 1-11 2010-2015,2020-2030
otext textapi no data, only specifications

Sectional features¶

feature description examples
book name of Bible book matthew mark 1john
